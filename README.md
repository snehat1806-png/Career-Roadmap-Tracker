# 🚀 Career Roadmap Tracker

A modern and responsive **Career Roadmap Tracker** built using **HTML, CSS, and JavaScript**. This application helps users organize learning goals, monitor progress, and stay motivated throughout their learning journey.

---

## 🌐 Live Demo

🔗 **Live Website:**  
https://snehat1806-png.github.io/Career-Roadmap-Tracker/

---

## 📌 Features

- ➕ Add new career goals
- ✅ Mark goals as completed
- 🗑️ Delete goals
- 🔍 Search goals instantly
- 📊 Live Dashboard
  - Total Goals
  - Completed Goals
  - Pending Goals
  - Overall Progress (%)
- 📱 Fully Responsive Design
- 🎨 Clean & Modern User Interface

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Snehat1806-png/Career-Roadmap-Tracker.git
```

### Open the Project

```bash
cd Career-Roadmap-Tracker
```

Finally, open **index.html** in your browser.

---

## 💡 How It Works

1. Enter your learning goal.
2. Click **Add Goal**.
3. Mark goals as completed.
4. Delete unwanted goals.
5. Search goals instantly.
6. Dashboard updates automatically.
